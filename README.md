# Inholland SMS service - Vue app
The Vue app for the Inholland SMS service.

### How to run (local)
1. Install the required dependencies:
```
$ npm install
```
2. Copy the .env.example file, and name it .env.

3. Run the app:
```
$ npm run dev
```

### How to run (Docker)
1. Run Docker Compose
```
$ docker-compose up -d
```
